# Life Ledger Skill

OpenClaw 生活记账 Skill

## 安装

```bash
curl -sSL https://raw.githubusercontent.com/Payton-Lee/life-ledger/main/skill/install.sh | bash
```

## 使用

在 OpenClaw 对话中直接说：
- `付款 25 午饭 店铺：面馆 ⭐⭐⭐⭐`
- `生成月报`
- `美食排行榜`

## 开发

```bash
git clone https://github.com/Payton-Lee/life-ledger.git
cd life-ledger/skill
pip3 install pytz
```
